2. Create PostgreSQL database in Render
3. Add secrets in Render dashboard:
   * DATABASE_URL
   * SECRET_KEY
   * OPENAI_API_KEY
4. Add render.yaml:

  - type: web
    name: backend
    env: python
    buildCommand: "pip install -r requirements.txt"
    startCommand: "gunicorn backend.wsgi"
    envVars:
      - key: DATABASE_URL
        fromDatabase:
          name: pg-careergpt
          property: connectionString

5. Enable auto-deploy on push
